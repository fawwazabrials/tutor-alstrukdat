# Cetak Persegi

Buatlah sebuah program bernama `cetakpersegi.c` yang mencetak sebuah persegi berdasarkan masukkan n, dimana panjang dari sisi persegi adalah `2n-1` dan berbentuk dengan pola 'O' dan 'X' secara selang-seling.

## Contoh
### Input 1
```
1
```
### Output 1
```
O
```


### Input 2
```
2
```
### Output 2
```
OXO
XOX
OXO
```

### Input 3
```
O
```
### Output 3
```
OXOXO
XOXOX
OXOXO
XOXOX
OXOXO
```
